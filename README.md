# 小艺APP智能体使用教程

## 一、什么是小艺智能体

小艺智能体是华为鸿蒙系统内置的 AI 助手，提供多种专业智能体，包括：

| 智能体 | 功能定位 |
|---|---|
| **小艺 Claw** | 个人 AI 助理，信息搜集、文档处理、内容创作 |
| **小艺帮帮忙** | 应用操控，购物、订票、下载等任务自动化 |
| **小艺文档助理** | AI 办公，文档摘要、格式转换、PPT生成 |
| **小艺深度解题** | 学习助手，拍照解题、作业批改、AI讲解 |

---

## 二、前置条件

### 支持机型

- 手机、平板（华为设备）

### 系统版本

| 功能 | 系统要求 |
|---|---|
| 基础功能 | HarmonyOS 5.0 及以上 |
| 小艺 Claw | HarmonyOS 6.0 及以上 |
| 小艺帮帮忙智能体 | HarmonyOS 6.0 及以上 |

### App版本

- 小艺 App 需更新至最新版本
- 部分功能需要特定版本（如小艺帮帮忙需要 11.3.10.301 及以上）

### 其他条件

- 需要连接网络
- 部分功能在小范围内测中

---

## 三、进入智能体

### 方式一：广场入口

```
打开小艺 App > 底部点击"广场"/"发现" > 找到对应智能体 > 点击进入
```

### 方式二：搜索入口

```
打开小艺 App > 底部点击"广场" > 右上角搜索 > 输入智能体名称 > 点击结果进入
```

### 方式三：对话列表

```
打开小艺 App > 在对话列表页中找到智能体 > 点击进入
```

### 方式四：快捷指令

```
打开小艺 App > 主对话界面输入框上方 > 点击智能体快捷指令
```

### 方式五：桌面快捷方式

```
进入智能体页面 > 右上角四个点 > 添加至桌面 > 确定
后续可通过桌面图标快速打开
```

## 四、热门智能体详解

### 4.1 小艺 Claw（个人 AI 助理）

小艺 Claw 是一款基于 OpenClaw 开源框架、融合小艺特有鸿蒙 Skills 能力打造的养成系个人 AI 助理。

#### 四大人设

| 人设 | 功能简述 | 推荐话术 |
|---|---|---|
| **信息猎手** | 信息搜集、热点追踪、知识查询与内容整理 | "帮我追踪最近机器人行业的热点新闻，整理成周会汇报要点" |
| **知心朋友** | 日常交流、答疑解惑、情绪陪伴与长期记忆 | "我最近有点焦虑，帮我梳理一下现在最该先做什么" |
| **办公搭子** | 效率提升、流程简化与任务协同 | "帮我把这个 PDF 提炼成三页汇报要点" |
| **创作天才** | 灵感启发、内容生成与表达优化 | "给我想 10 个适合机器人成长主题的短视频选题" |

#### 核心能力

- 信息检索与整合
- 文档理解与处理
- 任务拆解与协助
- 内容生成与优化
- 长时记忆与持续学习

---

### 4.2 小艺帮帮忙（应用操控）

专注应用操控，通过语音或文字指令自动完成购物、订票、下载等任务。

#### 支持场景

| 场景 | 支持应用 | 示例指令 |
|---|---|---|
| 购买商品 | 淘宝、京东、拼多多、华为商城 | "去华为商城购买一台 Pura80" |
| 搜索商品 | 淘宝、京东、拼多多、华为商城 | "去淘宝搜一件白衬衫" |
| 查订单 | 淘宝、京东、拼多多、华为商城 | "帮我查淘宝上待付款的订单" |
| 订机票 | 去哪儿旅行 | "帮我订一张从深圳去上海的机票，明天出发" |
| 视频下载 | 华为视频 | "帮我去华为视频下载魔童闹海" |

#### 学技能功能

- **看图学技能**：上传技能截图，小艺自动学习
- **联网学技能**：说出技能名称，小艺联网搜索学习
- **手把手教学**：手动演示操作步骤，小艺记录学习

---

### 4.3 小艺文档助理（AI 办公）

全能 AI 办公伙伴，支持文档阅读、创作、工具三大类功能。

#### 文档阅读

| 功能 | 说明 |
|---|---|
| 智能摘要 | 快速提炼 PDF/Word/Excel/PPT 核心内容 |
| 思维导图 | 自动生成结构化思维导图 |
| 文档问答 | 对文档内容精准提问 |
| 文档比对 | 两份文档差异比对 |

#### 文档创作

| 功能 | 说明 |
|---|---|
| Word 生成 | 智能润色/补写/排版 |
| 表格生成 | 设计生成、数据填充、信息整理 |
| PPT 生成 | 自定义页数、场景、模板 |
| 教案生成 | 根据教材生成教案文档 |

#### 文档工具

| 功能 | 说明 |
|---|---|
| 格式转换 | PDF↔Word、Word/PPT↔PDF、音视频→TXT |
| 网页转文档 | 微信公众号/小红书链接→Word/PDF |

#### 支持的文档类型

- PDF（最大 200M）
- Word（最大 150M）
- PPT（最大 150M）
- Excel（最大 30M）
- TXT（最大 15M）
- CSV、EPUB（最大 30M）

---

### 4.4 小艺深度解题（学习助手）

基于盘古-Deepseek 大模型，一站式提升学习效率。

#### 核心功能

| 功能 | 说明 |
|---|---|
| 拍照快答 | 拍单题、拍整页、跨页题快速解题 |
| 作业批改 | 自动批改作业，家长更轻松 |
| 错题本 | 错题管理、关联知识点出题 |
| AI 互动讲解 | 苏格拉底式追问，分步详解 |
| AI 辅导写作 | 语文/英语作文辅导 |
| 可视化解题 | 视频形式讲解难题 |
| 举一反三练 | 知识点精准练习 |

#### 操作方式

```
点击首页功能按钮 > 拍摄/上传题目 > 框选题目 > 确认 > 查看解答
```

## 五、使用技巧

### 1. 指令要完整

提供关键信息，明确指定应用和操作：

```
✅ 帮我到去哪儿买一张从上海到北京首都机场的机票，晚上6点后出发
❌ 帮我买机票
```

### 2. 善用桌面快捷方式

将常用智能体添加到桌面，一键直达。

### 3. 利用历史文件

智能体会保存上传和生成的文件，可随时查看历史记录。

### 4. 主动教学

对于小艺帮帮忙，可以主动教它新技能，让它更懂你的需求。

### 5. 选择合适的人设

小艺 Claw 提供四种人设，根据场景选择：
- 查资料 → 信息猎手
- 聊天解压 → 知心朋友
- 办公效率 → 办公搭子
- 创意写作 → 创作天才

## 七、注意事项

1. **仅支持华为设备**：小艺智能体为华为鸿蒙专属功能
2. **系统版本要求**：不同智能体有不同系统要求
3. **网络依赖**：大部分功能需要联网使用
4. **不支持的应用**：
   - 游戏类、金融类、拍照美化类
   - 浏览器、输入法、AI 助手
   - 微信等聊天社交应用
