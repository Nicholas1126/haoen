# 配置步骤详解

> 请按以下步骤完成环境配置，确保自动化测试正常运行。

---

## 步骤 1：安装基础依赖

执行以下命令安装项目依赖：

```bash
pip install adb-shell
pip install uiautomator2
```

验证安装：

```bash
adb version
```

---

## 步骤 2：配置设备连接

确保手机设备已连接并开启 USB 调试：

```bash
# 检查设备连接
adb devices
```

---

## 步骤 3：验证环境配置

完成上述配置后，使用项目自带的 GUI 操作脚本验证环境是否正常。

---

## 步骤 5：常见问题排查

### 问题 1：设备无法识别

```bash
adb kill-server
adb start-server
adb devices
```

### 问题 2：USB 调试未授权

在手机上确认 USB 调试授权弹窗。
